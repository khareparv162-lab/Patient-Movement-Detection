# Patient-Movement-Detection
This project is an AI-based patient monitoring system designed to detect and track patient movements in real-time, especially in hospital or elderly care environments. The system helps in identifying critical situations such as unusual movements, falls, or prolonged inactivity, enabling timely medical intervention.

🚀 Features
Real-time patient movement detection using computer vision
Pose estimation to analyze body posture and activity
Alerts for abnormal or risky movements
Lightweight and scalable for hospital use
Can be integrated with existing monitoring systems
🧠 Tech Stack
YOLO (object detection)
MediaPipe (pose estimation)
Python
OpenCV
PyTorch
📌 Current Progress
Basic movement detection implemented
Model setup using YOLO completed
Initial pose tracking using MediaPipe
Testing with sample video inputs
🔮 Future Scope
Fall detection and emergency alert system
Dashboard for live monitoring
Integration with IoT devices / hospital systems
Improved accuracy with custom-trained datasets
